# Práctica de Pruebas Unitarias en Python

Este proyecto es una práctica enfocada en el uso de **pruebas unitarias individuales** para verificar el correcto funcionamiento de operaciones aritméticas básicas.

Se implementan métodos de suma y división y se validan mediante pruebas que comprueban:
- Resultados correctos con datos válidos.
- Manejo de números negativos.
- Control de errores cuando se ingresan datos no numéricos.
- Validación de excepciones como la división entre cero.

El objetivo es aprender a estructurar pruebas, organizar casos de prueba y asegurar que cada función se comporte como se espera.
